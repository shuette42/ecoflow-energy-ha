# Code of Conduct

## Purpose

This project is a community-driven Home Assistant integration maintained by a single developer in their spare time. The goal is a welcoming, respectful space where contributors and users can collaborate effectively, regardless of background or experience level.

## Expected Behavior

- Be respectful and constructive in discussions, issues, and pull requests.
- Provide clear, actionable feedback. "This sensor shows the wrong value for my Delta 2 Max when charging above 80%" is helpful. "This is broken" is not.
- Assume good intent. Bugs happen, misunderstandings happen, and not everyone speaks English natively.
- Stay on topic. GitHub Issues are for bugs and feature requests.
- Credit others when building on their work or ideas.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language of any kind.
- Trolling, deliberately inflammatory comments, or sustained negativity.
- Publishing others' private information (addresses, credentials, device serial numbers) without consent.
- Demanding features or fixes with hostile or entitled language. This is a volunteer project.
- Spamming issues, discussions, or pull requests with unrelated content.

## Scope

This Code of Conduct applies to all project spaces: GitHub Issues, Pull Requests, Discussions, commit messages, and any other communication channel associated with this repository. It also applies when representing the project in external forums or communities.

## Enforcement

If someone violates this Code of Conduct:

1. **First occurrence** - a private reminder about the expected behavior.
2. **Repeated violations** - temporary or permanent restriction from project spaces.

Decisions are made at the maintainer's discretion, proportional to the severity of the behavior.

## Reporting

If you experience or witness unacceptable behavior, please reach out privately:

- **GitHub**: Contact [@shuette42](https://github.com/shuette42) via private message

Reports will be reviewed promptly and handled with discretion. All reports are treated confidentially.
