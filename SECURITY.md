# Security Policy

## Supported Versions

Only the latest release is supported with security updates.

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please use GitHub's private vulnerability reporting feature instead:

1. Go to the [Security tab](https://github.com/shuette42/ecoflow-energy-ha/security) of this repository
2. Click **Report a vulnerability**
3. Provide a clear description of the issue, steps to reproduce, and potential impact

I aim to acknowledge reports within 7 days. Fixes will be released as soon as possible depending on severity and complexity.

## Scope

**In scope:** Code within `custom_components/ecoflow_energy/` in this repository.

**Out of scope:**
- EcoFlow's API, cloud infrastructure, firmware, or mobile apps
- Home Assistant core or other third-party integrations
- Issues that require physical access to the device running Home Assistant

## Disclosure

Once a fix is released, the vulnerability will be disclosed in the release notes. Credit will be given to the reporter unless they prefer to remain anonymous.
